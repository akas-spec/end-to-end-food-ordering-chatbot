# ijju_eatery

An AI-powered food ordering chatbot built using FastAPI, Dialogflow, and PostgreSQL.

## Features
- Conversational food ordering
- Add / remove items from order
- Order tracking
- Dialogflow chatbot integration
- Clean frontend UI

##  Tech Stack
- Backend: FastAPI (Python)
- NLP: Dialogflow
- Database: PostgreSQL
- Frontend: HTML, CSS
- Webhook exposure: ngrok

## How to Run Locally

1. Activate virtual environment
2. Start backend:
   ```bash
   uvicorn main:app --reload
